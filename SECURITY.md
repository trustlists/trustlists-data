# Security Policy

Thanks for helping keep TrustLists Data secure.

## Supported scope

This repository mainly contains public dataset snapshots, schema/docs, and sync automation.

Please report:

- Vulnerabilities in repository automation (for example GitHub Actions misuse)
- Credential or token exposure in committed files
- Security issues in helper scripts

Please do not report ordinary public data inaccuracies here. Use issue templates for data corrections instead.

## How to report a vulnerability

Email: **security@trustlists.org**

Please include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested mitigation

## Response expectations

- Initial acknowledgment: within 3 business days
- Triage and impact assessment: as quickly as possible
- Remediation timeline: depends on severity and complexity

We ask that you avoid public disclosure until a fix or mitigation is available.
